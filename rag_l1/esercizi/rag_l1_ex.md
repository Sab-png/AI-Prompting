# Esercizio: Costruzione di un Sistema RAG per Documentazione Tecnica





## Obiettivo dell'Esercizio

Creare un sistema RAG (Retrieval-Augmented Generation) che permetta di interrogare in linguaggio naturale il **Manuale Gestionale Open**, un software gestionale ERP per PMI italiane.

## Obiettivi 
- Implementare le fasi di chunking, embedding e retrieval
- Integrare un LLM con un vector database
- Scrivere un prompt adeguato
